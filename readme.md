# Simple Car Rental db

CRUD api for car rental

## Getting Started


### Prerequisites

This project requires Node and Mongodb

### Installing

Install dependencies

```
npm install
```

Please change `dbServer` in the `app.js`

```
const dbServer = 'mongodb://db2.localnet:27017/stardata'
```

Start


```
npm start
```
